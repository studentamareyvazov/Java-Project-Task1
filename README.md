# Spring Boot Task 1 - Web Application

## Project Description
This is my first Spring Boot web application for Vistula University, demonstrating basic MVC pattern with Thymeleaf.

## Features
- Spring Boot 4.0.0 with Java 17
- Controller with @GetMapping mappings
- Thymeleaf template for dynamic content
- Static image serving

## Endpoints
- `GET /` - Returns plain text greeting
- `GET /greeting?owner=Name` - Returns HTML page with dynamic name

## How to Run
1. Clone repository: `git clone https://github.com/studentamareyvasov/Dava-Project-Task1.git`
2. Open in IntelliJ IDEA
3. Run `FirstProjectJavaSpringApplication.java`
4. Visit:
   - http://localhost:8080
   - http://localhost:8080/greeting?owner=Vistula
